# Library Management System (Java)

## 📚 Project Overview

This is a console-based Library Management System developed in Java. The project allows librarians to manage books and users to issue and return books through a simple menu-driven interface.

The application demonstrates the use of core Java concepts such as:

* Object-Oriented Programming (OOP)
* Arrays
* HashMap Collections
* Methods and Classes
* User Authentication
* Menu-Driven Programming
* Inventory Management

---

## 🚀 Features

### Librarian Module

* Secure Login
* Add New Books
* Delete Books
* Update Book Quantity
* View Available Books

### User Module

* Issue Books
* Return Books
* View Book Availability
* Maximum 2 Books per Student
